# 블로그를 시작하며

안녕하세요! 이 블로그는 **옵시디언(Obsidian)**으로 글을 작성하고, **GitHub Pages**로 발행하는 개인 포트폴리오 겸 블로그입니다.

## 워크플로우

이 사이트의 워크플로우는 다음과 같습니다:

1. 옵시디언에서 마크다운 파일 작성
2. `[[wiki-link]]`를 활용한 글 간 연결
3. Git 커밋 & 푸시
4. GitHub Actions가 자동으로 빌드 & 배포

자세한 내용은 [[obsidian-workflow]] 글을 참고하세요.

## 지식 그래프

이 블로그의 특별한 기능 중 하나는 [[knowledge-graph|지식 그래프]]입니다. 옵시디언의 그래프 뷰에서 영감을 받아, 글 간의 관계를 시각적으로 보여줍니다.

## 기술 스택

- **프레임워크**: Next.js (Static Export)
- **스타일**: Anthropic Research 톤의 커스텀 CSS
- **배포**: GitHub Pages + GitHub Actions
- **글 작성**: Obsidian
